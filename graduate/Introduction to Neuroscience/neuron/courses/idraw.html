A short idraw exercise

Exercise

1) Create an idraw file by running the HH Patch simulation in neurondemo, selecting the action potential graph on the Print&FileWindowManager into the paper rectangle (click right button on the appropriate window icon) and selecting the PrintToFile/Idraw item. Call the file temp1.id . Change the stimulus to 0.1 nA and create a second idraw file, temp2.id .

In this exercise we will combine the figures, make the lines darker, change fonts, add labels, etc.

2) Run idraw and use the file menu item to open temp1.id .

3) Use the file menu to "Import graphic" temp2.id . Select the two figures and overlay them by aligning the centers. Because they are not the same size, remove the one that was "open"ed and instead read it with "Import graphic".

4) Align the figures side by side ("Align/AbutRight") and remove the "v" characters (several Structure/Ungroup commands until the desired text items are isolated. Then select a text item and execute the Edit/Delete command. Regroup the figures so that all the component parts stay in the proper relation.

5) Thicken the lines using the Brush menu and use the Font menu to change the font to TimesRoman14.

6) Create some Text. "ms", "mV", "0.1nA", "0.25nA". Text can be duplicated by selecting and executing the Edit/Duplicate command. Move them to their proper locations. (View/GridOnOff and View/GravityOnOff can be helpful here.) Increase the size of the main labels with the Edit/PreciseScale command.
